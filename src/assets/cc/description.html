Easily create, edit, and post BuzzFeed style personality quizzes right in the editor! This plug-in is implemented as an editor block - it's as easy as adding a new paragraph.

  • Easy-to-use editor block interface
  • Beautifully minimal front-end styling, blends right into your theme!
  • Allow users to share quiz in 21 different ways
  • Optionally force users to share the quiz before seeing results
  • Create unlimited quizzes, even multiple per page (and post!)
  • Many more features to come in future releases

Not sure yet? Check out the free version of this plug-in on the WordPress plug-in directory!
